Measures of Position (Z-SCORE or Standard Score)

Z-score

General Formula: Z = (value - mean) / standard deviation

Population Formula: Z = (X - μ) / σ

Sample Formula: Z = (X - X̄) / s

It represents the number of standard deviations that a data value falls above or below the mean for a specific distirbution of values

If z-score = 0, the value is the same as mean

If z-score < 0, the value is below the mean

If z-score > 0, the value is below the mean
