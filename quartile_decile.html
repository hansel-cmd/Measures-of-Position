Measures of Position (Quartile and Decile)

Quartile

Divides the data set into 4 equal parts, separated Q1, Q2, Q3.

If the position is decimal, i.e, 4.25, you need to get the floor and the ceil of the position (4 and 5) and get the average of both values in that position.

Q1:

Formula: Q1 = (n + 1) / 4

Where n is the number of data values

Q2:

Formula: Q2 = (n + 1) / 2

Where n is the number of data values

Q3:

Formula: Q3 = 3(n + 1) / 4

Where n is the number of data values

Interquartile Range:

Formula: IQR: Q3 - Q1

The Interquartile Range (IQR) is defined as the difference between Q3 and Q1 and is the range of the middle 50% of the data.

The IQR is use to identify Outliers, and it is also used as a measure of variability in exploratory data analysis

Decile

Divides the data set into 10 equal parts, separated D1 to D9.

If the position is decimal, i.e, 4.25, you need to get the floor and the ceil of the position (4 and 5) and get the average of both values in that position.

Formula: Dk = k(n + 1) / 10

Where k is {1, 2, 3, 4, 5, 6, 7, 8, 9}

Where n is the total number of data
