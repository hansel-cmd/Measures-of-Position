Measures of Position (Percentile)

Finding the Percentile Rank given a score and the data set

Percentile:

Formula: Percentile = (number_of_values_below_X + 0.5) / total_number_of_values * 100

Divides the data set into 100 equal parts

Indicates the position of an individual in a group in terms of 100.

Finding a data value corresponding to a given percentile

C:

Formula: c = (n * p) / 100

Where n = total number of values

Where p = percentile

Step 1: Arrange the data set.

Step 2: Use the formula.

Step 3a: If c is not a whole number, round up to the nearest whole number. Starting at the lower value, count over to the number that corresponds to the rounded-up value.

Step 3b: If c is a whole number, use the value halfway between the cth and (c + 1)th when counting up from the lowest value.
